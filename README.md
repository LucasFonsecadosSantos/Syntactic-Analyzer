## Geting Started


## System Requirements
JDK 8.
Bash shell (if you want the init.sh script)

## How to compile

```
$ cd Syntactic-Analyzer
$ #For only compilation
$ chmod +x justCompile.sh
$ #For tests execution
$ ./init.sh test/<input_file_name>
```

## How to use

```
$ ./init.sh test/<input_file_name>
```
